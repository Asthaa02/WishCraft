# WishCraft

WishCraft is a responsive web application that allows users to create personalized digital greeting cards with customizable messages, occasions, and themes. Users can generate and download beautifully designed greeting cards as PNG images.

##  Features

- Personalized greeting cards
- Multiple themes
- Custom occasions
- Download as PNG
- Responsive design
- LocalStorage support

##  Tech Stack

- HTML
- CSS
- JavaScript
- LocalStorage
- Git
- GitHub
- Vercel
